
# Diabetes Prediction System using Machine Learning

## Overview

This project is a Diabetes Prediction System built using Machine Learning. The model is trained on the PIMA Indians Diabetes Dataset and uses a Support Vector Machine (SVM) classifier to predict whether a person is diabetic or non-diabetic based on medical attributes.

## Features

* Data preprocessing and analysis
* Feature standardization using StandardScaler
* Model training using Support Vector Machine (SVM)
* Model evaluation with accuracy score
* Predictive system for new patient data

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn

## Dataset

The project uses the PIMA Indians Diabetes Dataset, which contains medical diagnostic measurements such as:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

## Machine Learning Workflow

1. Data Collection and Analysis
2. Data Preprocessing
3. Feature Standardization
4. Train-Test Split
5. Model Training using SVM
6. Model Evaluation
7. Diabetes Prediction

## Results

The trained model achieves good prediction accuracy and can be used to classify whether a person is likely to have diabetes based on the provided health parameters.
